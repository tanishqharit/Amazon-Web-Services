# What is Cloud Computing?

## Simple Explanation

Cloud computing means **using someone else's computers over the internet** instead of buying your own.

Think of it like electricity:
- You don't build your own power plant at home
- You just plug into the wall and use electricity
- You pay for what you use

Cloud computing works the same way — but with computers, storage, and software.

---

## Before the Cloud

Before cloud computing, companies had to:

1. **Buy physical servers** (very expensive)
2. **Set up a server room** (need space, cooling, security)
3. **Hire people** to maintain the hardware
4. **Guess** how much capacity they would need
5. **Wait weeks or months** to set up new servers

If they bought too many servers → money wasted.
If they bought too few → website crashes.

---

## With the Cloud

Now companies can:

1. **Rent servers** from cloud providers (like AWS)
2. **Start in minutes** instead of weeks
3. **Scale up or down** based on need
4. **Pay only for what they use**
5. **Access from anywhere** with internet

---

## Real-World Analogy

| Concept | Old Way | Cloud Way |
|---------|---------|-----------|
| Watching movies | Buy DVDs | Netflix (stream) |
| Storing files | USB drives | Google Drive |
| Running software | Install on PC | Use in browser |
| Hosting a website | Own server | Rent from AWS |

---

## Key Takeaway

> Cloud computing = renting computing resources (servers, storage, databases) over the internet, on-demand, and paying only for what you use.
