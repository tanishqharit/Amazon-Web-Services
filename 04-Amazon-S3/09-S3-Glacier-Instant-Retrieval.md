# S3 Glacier Instant Retrieval

## What is it?

**S3 Glacier Instant Retrieval** is for **archive data** that you rarely access (once a quarter) but need **instantly** when you do.

---

## Key Features

| Feature | Detail |
|---------|--------|
| **Access** | Instant (milliseconds) — same as Standard |
| **Storage cost** | ~68% cheaper than S3 Standard |
| **Retrieval fee** | Higher than Standard-IA |
| **Min storage duration** | 90 days |
| **Best for** | Archive data needing instant access |

---

## When to Use

- Medical images accessed once per quarter
- Old analytics data needed for occasional reports
- Compliance data that might be requested suddenly

---

## Key Takeaway

> Glacier Instant Retrieval gives you archive-level pricing with instant access speed. 90-day minimum. Use it for rarely accessed data that must be available immediately.
