# S3 Glacier Deep Archive

## What is it?

**S3 Glacier Deep Archive** is the **cheapest** storage class in S3. It's designed for data that you almost **never** access and can wait **12-48 hours** to retrieve.

---

## Key Features

| Feature | Detail |
|---------|--------|
| **Access** | 12-48 hours |
| **Storage cost** | Cheapest in all of S3 |
| **Min storage duration** | **180 days** (6 months) |
| **Best for** | Long-term compliance archives |

---

## Retrieval Options

| Option | Time |
|--------|------|
| **Standard** | 12 hours |
| **Bulk** | 48 hours |

---

## When to Use

- Regulatory compliance (keep data for 7-10 years)
- Legal hold data
- Old financial records
- Data you might never access but must keep

---

## For the Exam

Keyword: **"Archive for 7 years"** or **"long-term retention"** → **Glacier Deep Archive**

---

## Key Takeaway

> Glacier Deep Archive is the cheapest S3 storage. Retrieval takes 12-48 hours. 180-day minimum. Use it for long-term compliance data you almost never access.
