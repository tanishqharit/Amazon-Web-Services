# Glue Studio

## What is Glue Studio?

**AWS Glue Studio** is a **visual interface** for creating, running, and monitoring Glue ETL jobs — without writing code.

It uses a **drag-and-drop** editor where you connect blocks to build your pipeline.

---

## How It Works

```
[S3 Source] → [Transform: Filter] → [Transform: Map] → [S3 Target]
```

You connect visual blocks and Glue Studio generates the PySpark code automatically.

---

## Key Features

- ✅ Visual drag-and-drop job editor
- ✅ Auto-generates PySpark code
- ✅ Built-in data preview
- ✅ Job monitoring dashboard
- ✅ Can edit generated code manually if needed

---

## When to Use

- You prefer visual tools over writing code
- Quick prototyping of ETL pipelines
- When you want to see the data flow visually

---

## Key Takeaway

> Glue Studio is a visual, no-code editor for building Glue ETL jobs. It generates Spark code automatically. Great for visual learners and quick prototyping.
